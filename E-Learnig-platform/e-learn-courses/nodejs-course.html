Welcome to Node js

Node.js is a powerful runtime environment that lets you run JavaScript on the server side. This tutorial will guide you through essential Node.js concepts with practical examples.

📌 What is Node.js?

  • Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine.
  • It allows you to run JavaScript code outside the browser — on the server.
  • Commonly used for building scalable network applications, APIs, and backend services.
  • Node.js files typically use the .js extension.

💡 Why Use Node.js?

Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. It's perfect for data-intensive real-time applications that run across distributed devices.

🧪 Node.js Example:


// Save this as app.js and run with: node app.js
const http = require('http');

http.createServer((req, res) => {
  res.writeHead(200, {'Content-Type': 'text/plain'});
  res.end('Hello Node.js!');
}).listen(3000);

To run this code, install Node.js and use the terminal command: node app.js
